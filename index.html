Silence trimming (Silero VAD)

Runs a small Silero VAD model locally in your browser to remove silent passages from the audio. Reduces Whisper hallucinations and saves API cost. Adds ~200–400ms of processing per recording (after first use; the model is cached).

Mono-model flow

Only applies when exactly one model is selected. Skips Send and copies the result to clipboard automatically.
